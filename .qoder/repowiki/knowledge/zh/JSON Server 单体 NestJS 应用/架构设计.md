根目录作为 NestJS 单仓入口，由 nest-cli.json 指定 sourceRoot 为 src，tsconfig.json 配置 @/* 路径映射与 ES2021/CommonJS 目标；package.json 提供 build/start/test 等统一脚本，所有子模块共享同一份依赖（@nestjs/*、typeorm、mysql2、jest 等）。运行时由 app_config 中的 main.ts 启动 NestFactory，并在 AppModule 中集中注册 core_infra（全局异常过滤器、统一响应拦截器、JWT 守卫、公开路由装饰器）以及 api_auth、api_user、api_article 三个领域模块，sql/init.sql 提供数据库初始化脚本，形成「核心横切 → 认证/用户/文章」三层依赖关系。
