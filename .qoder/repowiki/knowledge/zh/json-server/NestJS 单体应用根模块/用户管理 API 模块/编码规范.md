- DTO 统一使用 class-validator 装饰器（`@IsString`、`@IsEmail`、`@MaxLength`、`@IsOptional` 等）配合 class-transformer 的 `@Type` 进行请求体验证与类型转换。
- 分页查询继承公共 `PaginationDto`，并在 Service 层用 `skip/take/order` 组合 `findAndCount` 返回 `{ userList, total }` 结构。
- 对不存在资源的更新操作统一抛出 `BadRequestException('用户不存在')`，保持错误语义一致。
- 实体字段名采用驼峰命名并通过 `@Column({ name: 'xxx' })` 显式映射到数据库下划线列名（如 `ip_address`、`github_id`）。
