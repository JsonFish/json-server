根级 AppModule 作为唯一装配点：通过 TypeOrmModule.forRoot(mysqlConfig) 注入全局数据库连接，并以 imports 聚合 UserModule、AuthModule、ArticleModule；同时以 APP_FILTER/APP_INTERCEPTOR/APP_GUARD 三个全局 Provider 把 core/filter、core/interceptor、core/guard 下的 AllExceptionFilter、TransformInterceptor、AuthGuard 注册到整个应用管线，使所有业务模块共享统一的异常处理、响应包装与鉴权拦截。各 api/* 子模块仅暴露自身 Module，不直接依赖彼此，跨模块协作通过共享的 DTO（src/common/dto/pagination.dto.ts）与 config/*.ts 配置对象完成。
